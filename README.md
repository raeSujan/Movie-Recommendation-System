A simple Content-Based Movie Recommendation System built using Python and Machine Learning.
This project recommends movies based on similarity between movie features.

## 🚀 Features
Recommends similar movies based on selected title
Uses cosine similarity for recommendation
Simple and interactive interface
Built with Python

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## ⚙️ How to Run
Install dependencies:
pip install -r requirements.txt
Run the application:
streamlit run app.py
